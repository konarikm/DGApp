# Disc Golf Score Tracker

Tento projekt je závěrečnou prací do předmětů:
- AP7MT - Mobilní technologie
- AP7PD - Pokročilé databázové systémy

Aplikace umožní uživatelům:
- Vyhledávat/Prohlížet discgolfová hřiště
- Zaznamenat skóre na jednotlivých jamkách na vybraném hřišti
- Prohlížet skórkarty a zobrazit základní statistiky hráče

## Skruktura projektu
- backend/ — Node.js + Express API spolu s MongoDB (Atlas)
- mobile/ — Android aplikace napsaná v Kotlin za použití Jetpack Compose
